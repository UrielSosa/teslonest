# TESLOAPI

1. Clonar el repo
2. Instalar las dependencias
```
yarn install
```
3. Clonar el archivo ```.env.example``` y renombrarlo a ````.env```
4. Cambiar las variables de entorno
5. Levantar la db
```
docker-compose up -d
```
6. Levantar la api
```
yarn start:dev
```
